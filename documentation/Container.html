Class: Container

Container(id, size, allowLengthChange)

The Container class provides a small set of functions,
allowing you to easily tile your Webpage without the need to
have the HTML-shape of the container system in mind all the time.

Furthermore, the functions have been written to be able to eventually support resizing and moving of containers via some UI (which would of course still have to be programmed).

Methods

moveTo(orientationName, …targetKeys)

Moves this Container next to another container.
This (and Container.setRoot) is the preferred way of placing a Container somewhere (or moving it somewhere else)

Parameters:
Name Type Attributes Description
orientationName 'left' | 'right' | 'up' | 'down'

where to place this Container relative to the new one
targetKeys key <repeatable>

a key to another Container.
If multiple are provided, it takes their smallest common Parent.
