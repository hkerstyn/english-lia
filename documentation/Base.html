(static) insert(mode, parentKey, elementKeys)

Inserts one or multiple elements as children of some parent.
if 'mode' is equal to

  • "set": the parent is being cleared of previous children
  • "make": the process gets aborted if parent already has children
  • "add": the new children are simply added to the existing ones
Parameters:
Name Type Description
mode string

either "set", "make", or "add"
parentKey key

where to insert the elements
elementKeys key

the elements to insert somewhere
