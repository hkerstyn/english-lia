Class: LulFunctions

LulFunctions

A PseudoClass. Provides genFunctions to create beautiful UI.
Before you can use any though, initialize the Library using the LulConfig class first.

ATTENTION: Most functions here use the arg input scheme

(static) genCollapsible(innerId, direction, toggle, hover, functions)

Generates a box, that can be collapsed and expanded via javascript.
Uses arg

Parameters:
Name Type Description
innerId string

the value specified here can be used
as key to insert other elements into the collapsible
direction 'row' | 'column'

whether the box should collapse
horizontally or vertically
toggle key

(optional) a key to an html-element that
gets set to trigger an expansion or collapse of this box
when clicked
hover key

(optional) a key to an html-element that
gets set to trigger an expansion or collapse of this box
when hovered.
functions Object

(optional) some object that gets assigned three new properties:

  • expandFunction: call this to expand this specific box
  • collapseFunction: call to collapse
  • toggleFunction: automatically expands or collapses the box
